Gene enrichment analysis with Enrichr¶

In this tutorial we show the use of Enrichr (Chen et al. 2013, Kuleshov et al. 2016) to carry out the enrichment analysis of a list of differtially expressed genes, which we obtained from the RNA-Seq analysis of two NGS Human breast cancer data sets.

Running a Galaxy Docker container using Kitematic¶

In this tutorial we show how to set up your system in order to run the Galaxy RNA workbench (Grüning et al. 2017, Fallmann et al. 2019) as a Docker container, using the Kitematic desktop application.
