de.STAIR¶

See project details at de.STAIR project page

Tools¶

  • Demultiplexing flexbar

  • Quality check with good old fastqc or qualimap

  • Pre-processing fastx - alternatively trimmomatic or trimgalore

  • Mapping kallisto

    • See for reference genomes: RefSeq or Ensembl

  • DE Analysis and statistics with kallisto sub-sequent tool sleuth

    • Preferred setup for larger data sets, because kallisto is incredibly fast

    • Sleuth needs R/Rstudio

    • Alternatively using traditional mappers like Star or Hisat2

    • TRAPLINE only for showcases =P
